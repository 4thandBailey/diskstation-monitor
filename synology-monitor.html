Scoped API Tokens
Per-appliance scoped tokens — DSM 7.2+ API Portal · least privilege
🔐
Each appliance issues a narrowly-scoped API token from DSM Control Panel → Security → API Portal. The token grants only the File Station endpoints required for polling — SYNO.FileStation.Info, List, MD5, DirSize, CheckPermission, VirtualFolder, and BackgroundTask. No administrative credentials are stored in this portal or on Railway. Tokens can be revoked per-appliance without affecting DSM login credentials.

DSM Notification Relay
Route alerts through a managed appliance's built-in SMTP
💡
DiskStation Monitor delegates outbound email to a designated Synology appliance — the same way DSM sends its own disk failure and update notifications via Control Panel → Notification → Email. The appliance handles SMTP authentication, TLS negotiation, and retry queuing. No separate mail server or credentials stored in this portal.
Choose which appliance forwards alert emails. Prefer a device that is always online — typically your primary or backup NAS.
